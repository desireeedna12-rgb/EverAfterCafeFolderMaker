# Week 6 Folder Maker App

Café Name: Ever After Cafe Folder

## Purpose
Creates the folder structure for the Ever After Café project.

## How To Run
1. Run the console program.
2. Enter the café name.
3. Press Enter.
4. The folders and README.txt file will be created.

## Folder Structure Created
- Menu
- Hours
- Images
- Docs
- README.txt

## Story to App Connection
Ever After Café is a Disney-inspired coffee shop where guests can browse the menu, view store hours, and explore magical beverages inspired by classic Disney stories.
